# Webpage Layout Assignment

This project is a simple webpage designed using **HTML, CSS, and JavaScript** as part of an internship assignment.

## Project Description
The objective of this project is to create a basic webpage layout including the following components:

- Header (Logo on left, social links on right)
- Menubar (Navigation menu)
- Sidebar (Custom data and links)
- Main Content Section
- Three content boxes (Title, Image, Description)
- Footer section
- Day and Night mode toggle using JavaScript

## Technologies Used

- HTML
- CSS
- JavaScript

## Webpage Structure

The webpage contains the following layout:

Header  
Menubar  
Sidebar (Latest News, Announcements, Updates)  
Main Content (Three Service Boxes)  
Footer

## Features

- Clean webpage layout
- Navigation menu
- Sidebar with useful links
- Content boxes with image and description
- Day / Night mode toggle
- Simple and responsive design
